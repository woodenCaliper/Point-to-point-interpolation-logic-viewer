# Point-to-point-interpolation-logic-viewer

2D の点から点への経路生成ロジックを比較する単一HTMLビューアです。

## 使い方
1. `index.html` をブラウザで開く。
2. 左ペインでアルゴリズムと入力値（Start / End / Path）を設定する。
3. 中央ペインで経路を確認する。
4. 右ペインで表示設定と生成結果を確認する。

## 単位
- 位置・長さ: `mm`
- 角度: `° (degree)`
- 曲率: `1/mm`

## 実装済みアルゴリズム
- 直線補間 (Linear)
- Dubins Path
- CC-Dubins Path（連続曲率を意識した近似）


## 中央ペイン操作
- ホイールでズーム（視点の近接/遠景）
- 左ドラッグで視点パン
- 左下に +X / +Y の固定軸アイコン表示
- グリッド分解能は 1/5 系列（1, 5, 10, 50, ...）で自動調整

# Point-to-point-interpolation-logic-viewer

2D の点から点への補間ロジック（Linear / Dubins / CC-Dubins）を比較できる、単一HTMLビューアです。

## 使い方
1. `index.html` をブラウザで開く。
2. 左ペインでアルゴリズムと入力（Start / End / Path）を調整する。
3. 中央ペインでパスを確認する（ホイールズーム / 左ドラッグパン）。
4. 右ペインで候補表示ON/OFF、最短のみ表示、統計情報を確認する。

## 主な仕様
- 単位: 位置/長さ = `mm`, 角度 = `°`, 曲率 = `1/mm`
- 座標系: 右が `+X`, 上が `+Y`
- 入力は即時反映（生成ボタンなし）
- 数値入力はスピンボタンで調整可能（mm項目は10mm刻み、角度項目は5°刻み）
- 角度入力 `θ` は ±180° を超えると循環（ラップ）する
- NG入力は赤枠表示し、入力が有効になるまでパス生成を停止
- グリッドは常時表示（1/5 系列で自動調整）し、現在の `mm` 刻みを中央ペインに表示
- 描画サンプリングは入力サンプル間隔より細かく（約5倍）分割して生成
